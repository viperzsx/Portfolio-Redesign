Imperial BattleBots (Python Game)

This project simulates mercenary negotiations in a zero-sum strategy game using modular Python. I built this as part of my programming coursework using two separate Python files:

  • ImperialBattleBotsMod.py – Handles functions like name generation, salary calculations, and input validation.
  • ImperialBattleBotsGame.py – The main loop that runs the game using user input and logic from the module file.

Features include randomized name/region generation, loot-based salary formulas, and total stat summaries. This project helped me strengthen my Python fundamentals like loops, conditionals, math logic, and modular programming.
